# Show KJN Color Palette

Displays the KJN color palette visually using base R plotting.

## Usage

``` r
show_kjn_colors(labels = TRUE)
```

## Arguments

- labels:

  Logical. If TRUE (default), shows color names and hex codes.

## Examples

``` r
# Show the palette
show_kjn_colors()


# Show without labels
show_kjn_colors(labels = FALSE)

```
