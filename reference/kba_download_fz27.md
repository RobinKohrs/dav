# Alle KBA FZ27-Excel-Dateien herunterladen

Alle KBA FZ27-Excel-Dateien herunterladen

## Usage

``` r
kba_download_fz27(
  dir = "kba_fz27",
  overwrite = FALSE,
  verbose = TRUE,
  page_url = .KBA_FZ27_PAGE_URL
)
```

## Arguments

- dir:

  Zielverzeichnis.

- overwrite:

  Überschreiben?

- verbose:

  Fortschrittsmeldungen.

- page_url:

  URL der Übersichtsseite.

## Value

Unsichtbarer Vektor mit Pfaden.
